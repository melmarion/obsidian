### PROMPT 5C: Performance Optimization

```

1. Character static image:

   - Size: Max 500KB
   - Dimensions: 2x actual display size (for retina)

2. Character video:
   - Compress to under 5MB
   - Use video poster for instant display
   - Consider disabling on slow connections

3. Cloud PNGs:
   - Compress to ~50KB each
   - Use CSS sprites if multiple clouds

ANIMATION PERFORMANCE:
- Use `transform` and `opacity` only (GPU accelerated)
- Avoid animating `width`, `height`, `top`, `left`
- Use `will-change` sparingly:
  .character-video { will-change: transform; }

LOADING STRATEGY:
1. Critical CSS inline in <head>
2. Defer non-critical animations
3. Lazy load video: <video loading="lazy">
4. Preload character image: <link rel="preload" as="image">

ACCESSIBILITY:
- Add `prefers-reduced-motion` check:
  @media (prefers-reduced-motion: reduce) {
    * { animation: none !important; }
    .cloud-1, .cloud-2, .cloud-3 { animation: none; }
  }

```
