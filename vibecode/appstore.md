## <a name="pre-launch"></a>🔍 PRE-LAUNCH: Market Research & Validation

### Trending Keyword Research (Darius Method)

```
I want to build a profitable app using the trending keyword method. Help me:

1. Research keywords that are:
   - Recently trending (increased search volume in last 7-90 days)
   - Have low/no competition
   - Generic terms (not brand-specific)
   - Search volume between 20-40+

2. For each keyword found, analyze:
   - Historical search volume trends (look for steady increases, not viral spikes)
   - Existing apps ranking for this term
   - Whether top apps use the keyword in their title
   - App complexity/build time required
   - Monetization potential (LTV estimate)

3. Create a priority list of top 5 keywords with:
   - Keyword name
   - Current search volume
   - Competition level (1-10)
   - Estimated build time
   - Revenue potential rating

My niche interests: [LIST YOUR INTERESTS OR NICHES]

Output format: Table with clear go/no-go recommendations for each keyword

```

### Supply & Demand Validation

```
I'm considering building an app in [NICHE]. Before I build anything, help me validate demand:

1. Identify what people are actively searching for in this niche
2. Find gaps where demand exists but supply is poor/missing
3. Analyze top 10 apps in this space:
   - What are their ratings/reviews saying?
   - What features do users complain are missing?
   - What do 1-star reviews consistently mention?
4. Determine if this is a trending-up or trending-down niche
5. Calculate estimated TAM (Total Addressable Market)

Based on this analysis, give me a clear YES/NO on whether to proceed, and if yes, what the minimum viable feature set should be.

```

## <a name="aso"></a>📱 ASO (App Store Optimization)

### Title & Subtitle Optimization

```
Optimize my app store title and subtitle for maximum ASO impact:

App name: [YOUR APP NAME]
Core function: [WHAT IT DOES]
Target keywords (in priority order): [KEYWORD 1], [KEYWORD 2], [KEYWORD 3]

Rules:
- Title max 30 characters
- Subtitle max 30 characters
- Primary keyword MUST be at the beginning of title
- Should feel natural, not keyword-stuffed
- Communicate clear value proposition

Provide 5 variations, ranked by:
1. ASO strength (keyword placement)
2. User clarity (will they understand what it does?)
3. Conversion potential (will they download?)

For each variation, explain the strategic choices made.

```

### Keyword Research From Zero

```
I'm launching [APP DESCRIPTION]. I need comprehensive keyword research:

1. Primary keywords (5-10):
   - What users directly search for
   - High relevance, medium-high volume
   - Achievable ranking potential

2. Secondary keywords (10-20):
   - Related terms
   - Long-tail variations
   - Lower competition

3. Negative keywords to avoid:
   - Brand names
   - Overly competitive terms
   - Misleading/irrelevant terms

4. Competitor keyword analysis:
   - Top 5 apps ranking for my primary keywords
   - What keywords are they using that I'm missing?
   - Where are the gaps I can exploit?

Format as a keyword strategy doc with:
- Priority tier (1-3)
- Search volume estimate
- Competition level
- Where to use it (title/subtitle/description/keywords field)

```

## <a name="tiktok"></a>🎬 TIKTOK & SHORT-FORM CONTENT

### Hook, Demo, Reveal Format (Kyle Method)

```
Create 10 TikTok video scripts using the Kyle Fowler format:

App: [YOUR APP NAME]
Core function: [WHAT IT DOES]
Target user: [WHO IT'S FOR]

Video format requirements:
- First 3-5 seconds: Show desirable outcome or intriguing setup
- Next 5-10 seconds: Demo the app in-hand scanning/analyzing
- Final 3-5 seconds: Reveal the result with suspense
- Total length: 15-30 seconds max
- No talking (or minimal voiceover)
- Text hook on screen

For each script provide:
1. Opening hook (what's shown + text on screen)
2. Demo sequence (shot-by-shot)
3. Result reveal (what's shown)
4. Suggested audio/sound
5. Why this hook will work (psychology behind it)

Variations should test different:
- Hooks (question vs statement vs shock value)
- Product angles
- Result types
- Suspense levels

```

### Calm Horizontal Format (Ununice Method)

```
I want to stand out from typical TikTok content by using the "calm" format. Create 5 video concepts:

App: [YOUR APP]
My personality/vibe: [DESCRIBE YOURSELF]
Story angle: [BUILD IN PUBLIC / FOUNDER JOURNEY / USER STORIES / etc]

Video format:
- Horizontal video on vertical platform
- Minimal editing
- Longer form (60-90 seconds)
- Just talking to camera authentically
- Still has hook, but softer
- Show vulnerability/behind-the-scenes

For each concept:
1. Opening line (hook)
2. Story arc (beginning, middle, end)
3. Key moments to hit
4. Ending CTA (subtle)
5. Shooting notes (where, how, lighting)

These should feel like I'm talking to a friend, not selling.

```

## <a name="viral"></a>🚀 VIRAL MARKETING MESSAGES

### Finding Your Viral Hook

```
Help me identify the viral marketing message for [APP NAME]:

App function: [WHAT IT DOES]
Target user: [WHO IT'S FOR]
Key features: [LIST FEATURES]

Analyze:
1. What's the core human desire this taps into?
   - Sexual attraction
   - Social status
   - Self-improvement
   - Entertainment
   - Fear/safety
   - Belonging
   - Curiosity

2. What's the transformation, not the feature?
   - Bad state → Good state
   - What emotional journey happens?

3. Create 10 viral marketing angles:
   - Each focused on a different psychological trigger
   - Written as a hook (first 5 seconds of video)
   - Explains why each could go viral

4. For the top 3 angles:
   - Full video concept
   - Why it will resonate
   - How to test it
   - Similar viral examples

Remember: No one cares about the product. They care about what it does for THEM.

```

### Niche Market Research Deep Dive

```
I'm building [APP DESCRIPTION]. Do comprehensive market research to find viral content formats:

Research process:
1. **Platform:** [TIKTOK / INSTAGRAM / YOUTUBE]
2. **Search terms:** [RELATED TO MY NICHE]
3. **Filter:** Top performing (most likes/views)
4. **Time range:** Last 90 days

For top 20 videos, analyze:
- What made it go viral? (break down the formula)
- Hook structure
- Video format
- Length
- Psychological trigger
- CTA method (if any)
- Comment section themes (what resonated)

Then:
- Identify 3-5 repeatable patterns
- Create templates I can adapt for my app
- Provide 5 example scripts using these patterns
- Explain how to make them feel native to my app (not forced)

```

### Anti-Pattern Strategy (Ununice Method)

```
Everyone in [MY NICHE] is doing [X TYPE OF CONTENT]. Help me do the opposite and stand out:

Current landscape:
- What everyone is doing: [DESCRIBE COMMON PATTERNS]
- Why it's oversaturated: [EXPLAIN]
- My app: [DESCRIPTION]

Create an anti-pattern content strategy:
1. Identify what's being overdone
2. Find the opposite approach for each element:
   - If they use fast cuts → I use slow, single-take
   - If they use loud music → I use silence or calm music
   - If they use vertical → I use horizontal
   - If they front-load CTA → I delay it until end
   - If they use text-heavy → I use minimal text

3. Design 5 content formats that:
   - Stand out through contrast
   - Still maintain hooks and conversion elements
   - Feel refreshingly different
   - Are repeatable

4. Psychology explanation:
   - Why standing out matters more than best practices
   - When to use this (new market vs saturated)
   - Risk/reward analysis

```
